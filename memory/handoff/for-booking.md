## 🔴 Critical 수정 항목

### [BK-01] 예약 하드코딩 가격 불일치

**파일**: [src/pages/admin/bookings.tsx:6](../../src/pages/admin/bookings.tsx#L6)
```typescript
{ id: 'B2040', name: '김민준', phone: '010-1234-5678',
  type: '3번 데크 + 불멍', date: '2026-04-12', time: '18:30',
  guests: 6, price: 189000, status: 'confirmed' }
```

**문제**: 2026-04-12는 **일요일(주말)**. 6명 + 불멍 포함 기준 정확한 금액은:
- 주말 기본(4인): 159,000
- 추가 2명: +40,000
- 불멍: +30,000
- **합계: 229,000원**

하드코딩 값 `189000`은 주말·추가인원 요금 미반영.

**수정**: `price: 229000` 으로 변경 (또는 데모 데이터임을 감안하여 계산식과 일치하는 다른 조합으로 재구성)
