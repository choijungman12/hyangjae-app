## ⚠️ 핵심 정책 변경

**고추냉이 스마트팜 = 가이드 투어 전용** (수확 불가)

지금까지 "와사비 수확 25,000원" 체험 상품이 있었지만, 사업주 확정 결정으로 변경:
- 고추냉이는 재배 기간이 18~24개월로 길어 수확량이 매우 제한적
- 데크 예약에 **투어만 기본 포함** (약 20분, 웰컴 시식)
- 와사비 수확 체험 상품(25,000원)은 **"와사비 강판 체험 5,000원"** 으로 대체
  - 투어 중 직접 강판으로 갈아보는 짧은 추가 옵션

## 📋 작업 1 — `src/pages/crop-detail/page.tsx` 고추냉이 섹션

### [SF-05] "수확 불가 · 투어 전용" 명시

고추냉이(`wasabi`) 섹션에 다음 배너 추가 (기존 JSX 구조 유지):

**위치**: 페이지 상단 또는 "재배 정보" 카드 내부

**내용**:
```
🌿 향재원 고추냉이 스마트팜은 가이드 투어 전용 공간입니다
• 재배 기간이 18~24개월로 길어 일반 수확은 제공하지 않습니다
• 약 20분 가이드 투어 + 웰컴 와사비 시식 포함
• "와사비 강판 체험" (+5,000원) 옵션으로 직접 갈아볼 수 있습니다
```

**디자인**: 기존 카드 스타일(`bg-white/80 backdrop-blur-xl rounded-3xl ...`) 유지, 안내 색상은 blue 또는 emerald 계열.

## 📋 작업 2 — `src/pages/cultivation-calendar/**` 또는 `growing-guide/**`

### [SF-06] 고추냉이 재배 기간 정보 명확화

기존 가이드에 "재배 기간: 18~24개월" 이 이미 있을 것. 없으면 추가. 있으면 그대로 유지.

사용자가 "왜 수확이 불가능한지" 납득할 수 있도록 본문 카피에 **한 문장 추가**:

> "고추냉이는 정식 후 18~24개월의 긴 재배 기간이 필요하여, 향재원에서는 대량 수확이 아닌 **소량 직판·투어 시식** 형태로만 운영됩니다."
