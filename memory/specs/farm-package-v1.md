## 1. 포지셔닝 전환

향재원은 단순 글램핑 데크가 아닌 **"데크별 전용 텃밭을 가진 파머스 글램핑"**로 재정의됩니다.

- 경쟁군: 하남 비욘더팜(공간대여 저가 모델), 과천·분당 프리미엄 글램핑
- 향재원 USP(타 업체 복제 불가):
  1. 강남권 30분 내 입지 (양재동)
  2. 진짜 스마트팜 (고추냉이 실제 재배, 150평)
  3. **데크별 전용 텃밭 수확 체험** (국내 유사 카테고리 없음)
  4. 콜키지 프리 (비욘더팜 벤치마크)

## 2. 기능 1 — 데크별 텃밭 수확 (Harvest Package)

### 2-1. 정책
| 항목 | 값 |
|---|---|
| 기본 제공량 | **1팀당 300g** (4인 기준) |
| 인원 추가 시 | 1명당 **+75g 자동 증량** (별도 과금 X, 추가 인원 요금에 포함) |
| 제공 작물 | 상추·로메인·양상추·깻잎·딸기 (시즌별 로테이션, 2~4종) |
| 체험 방식 | 해당 데크 전용 텃밭에서 직접 수확 |
| 추가 과금 없음 | 기본 요금(139K/159K)에 **전면 포함** |

### 2-2. 수확량 예시 (체감 환산)

|---|---|
| 상추 / 로메인 | 쌈 약 40~50장 |
| 깻잎 | 약 60장 |
| 딸기 (시즌) | 중간 크기 15~20알 |

### 2-4. 시즌 로테이션

## 5. 기능 4 — 파트너 제휴 모델 (고기 조달)

### 5-1. 개념
- 비욘더팜 → 샘터가든(10분 내 배달) 모델 벤치마크
- 양재동 반경 1~2km 내 정육점·한식당 1~2곳 제휴
- 향재원은 **중개만** (주문·결제·배송 관여 X → 법적 리스크 0)
- 예약 확인 메일·앱 내 안내에 "파트너 상점 배달 가능" 섹션 노출

### 5-2. 데이터 구조 (`partnerStores`)
```ts
type PartnerStore = {
  id: string;
  name: string;              // 실명 또는 "(제휴 협의 중)"
  category: 'butcher' | 'restaurant' | 'grocery';
  phone: string | null;      // 미확정 시 null
  deliveryTimeMin: number;   // 예상 배달 시간(분)
  distance: string;          // 예: "도보 5분", "1.2km"
  notes: string;              // 운영 시간, 최소 주문 등
  confirmed: boolean;        // 실 제휴 확정 여부
};
```

**Sprint-03 초기값 (플레이스홀더)**:
```ts
export const PARTNER_STORES: PartnerStore[] = [
  {
    id: 'partner-butcher-1',
    name: '(제휴 협의 중 · 정육점)',
    category: 'butcher',
    phone: null,
    deliveryTimeMin: 15,
    distance: '—',
    notes: '양재동 반경 1km 내 정육점 제휴 협의 예정',
    confirmed: false,
  },
  {
    id: 'partner-restaurant-1',
    name: '(제휴 협의 중 · 한식당)',
    category: 'restaurant',
    phone: null,
    deliveryTimeMin: 20,
    distance: '—',
    notes: '반찬·국·구이류 배달 협의 예정',
    confirmed: false,
  },
];
```

- 프론트엔드는 `confirmed: false`인 경우 "협의 중" 배지 표시
- 실제 제휴 체결되면 값만 교체 (구조 변경 불필요)
