## 2. 기능 1 — 예약금 (Deposit)

### 2-1. 정책
| 항목 | 값 |
|---|---|
| 예약금 비율 | **총액의 30%** (반올림, 100원 단위) |
| 결제 수단 | **네이버페이** (Sprint-02는 Mock UI) |
| 잔액 결제 | **현장** (카드·현금·페이 자유) |
| 환불 | 기존 취소 정책 동일 적용 (2일 전 100% / 1일 전 50% / 당일 0%) |
| 미결제 홀드 | 결제 미완료 시 **10분 후 자동 취소** |

### 2-2. 계산 예시
| 시나리오 | 총액 | 예약금(30%) | 현장 잔액 |
|---|---|---|---|
| 평일 4인 기본 | 139,000 | **41,700** | 97,300 |
| 주말 6인 + 불멍 | 229,000 | **68,700** | 160,300 |
| 와사비 수확 체험 2인 | 50,000 | **15,000** | 35,000 |

### 2-3. UX 흐름
```
[예약 단계 4/5: 결제]
┌─────────────────────┐
│ 총 금액     229,000원│
│ 예약금(30%)  68,700원│ ← 네이버페이로 결제
│ 현장 결제   160,300원│ ← 당일 현장에서
├─────────────────────┤
│ [네이버페이로 예약금 결제] │
└─────────────────────┘
```

### 2-4. Mock 결제 화면 (Sprint-02)
- 네이버페이 로고 + 결제 버튼
- "결제 중…" 3초 스피너
- 성공 페이지로 리다이렉트 (실 PG 연동 X)
- 쿼리스트링: `?mockPay=true&amount=68700&orderId=...`
- 실 연동은 Sprint-03 이후

### 2-5. 환불 시 예약금 처리
- 2일 전 취소 → 예약금 100% 환불 (네이버페이 취소 API, Sprint-03)
- 1일 전 취소 → 예약금의 50%만 환불
- 당일 취소/노쇼 → 예약금 **전액 몰수**

## 3. 기능 2 — 스탬프 리워드

### 3-1. 정책
| 항목 | 값 |
|---|---|
| 적립 단위 | **예약 1건당 1개** (요일·금액 무관, 평일/주말/체험 동일) |
| 적립 트리거 | 예약 **이용일 경과 + status='done'** 일 때 자동 |
| 목표 개수 | **10개** |
| 10개 달성 보상 | **향재원 고추냉이 선물 세트** (`wasabi-gift-set`, 89,000원 상당) |
| 쿠폰 유효 기간 | 발급 후 **6개월** |
| 회원 탈퇴 시 | 스탬프 + 미사용 쿠폰 **리셋** (복구 불가) |
| 적립 취소 조건 | 예약 취소·노쇼 시 스탬프 **적립 안 됨** |

### 3-2. 데이터 구조 (부킹 파트 구현 참고)
```typescript
// src/lib/stampStore.ts (개발자 A 신규)
type StampState = {
  userId: string;
  total: number;        // 누적 개수
  current: number;      // 현재 스탬프판 (0~10)
  rewards: Reward[];    // 발급된 쿠폰 이력
};

type Reward = {
  id: string;
  type: 'wasabi-gift-set';
  issuedAt: string;     // ISO date
  expiresAt: string;    // +6개월
  status: 'available' | 'used' | 'expired';
  usedAt?: string;
};
```

### 3-3. UX 흐름
```
[프로필 > 스탬프판]
┌─────────────────────────────┐
│ 향재원 스티커 🎟️ 7/10        │
├─────────────────────────────┤
│ ✅ ✅ ✅ ✅ ✅ ✅ ✅ ⬜ ⬜ ⬜ │
├─────────────────────────────┤
│ 10개 달성 시:               │
│ 🎁 향재원 고추냉이 선물 세트   │
│    (6개월 유효)              │
└─────────────────────────────┘

[10개 달성 시 모달]
┌────────────────────────────┐
│ 🎉 축하합니다!              │
│                            │
│ 향재원 고추냉이 선물 세트    │
│ 쿠폰이 발급되었습니다!       │
│                            │
│ 유효기간: 2026-10-12까지   │
│                            │
│ [쿠폰 확인하기]             │
└────────────────────────────┘
```

### 3-4. 쿠폰 사용 흐름
- 쿠폰 발급 → 프로필 > 내 쿠폰함에 보관
- 사용 시 → 쇼핑몰에서 `wasabi-gift-set` 주문 시 "쿠폰 적용" 버튼
- 적용 → 100% 할인 처리, 배송비만 결제 (추후 정책)
