향재원(HYANGJAEWON)은 서울 서초구 양재동 178-4(536평)에서 운영되는 **스마트팜 공간대여 서비스** 앱.

**Why:** 단순 글램핑이 아닌 "농업 체험 공간대여"가 핵심 USP. 고추냉이 실제 재배(150평) + 데크별 전용 텃밭 수확 체험이 강남권 30분 내 유일한 카테고리로 포지셔닝.

**How to apply:**
- 사업자: 팜스마트 / 대표 최정만 / 010-4929-0070
- 오픈 예정: 2027년 6월, 기준 날짜 2026-04-12
- 4개 파트 협업: 독스(기획/QA, 메인 조율자) + 부킹(예약/이커머스) + 스마트팜(IoT/농업데이터) + 디자인(UI/UX)
- 단일 저장소 main 기반, 역할별 브랜치(`docs/review`, `feature/booking`, `feature/smartfarm`, `design/ui-refactor`)
- 독스가 `memory/handoff/*.md` 지시서로 다른 파트에 작업 분배, QA 후 main 병합 책임
- 메인 시설: 메인 하우스 스마트팜 150평(고추냉이 전용, 가이드 투어만), 스마트팜 체험 데크 8개(데크별 전용 텃밭 포함, 4인 기준 최대 8인)
- 데크 텃밭 작물: 상추·로메인·깻잎·양상추·딸기 시즌 로테이션
- 지원 작물 14종(스마트팜 데이터): 상추·케일·시금치·깻잎·토마토·딸기·파프리카·오이·방울토마토·무·당근·고추냉이·바질·민트
