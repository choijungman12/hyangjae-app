향재원의 모든 카피·코드·문서에서 "글램핑", "Glamping", "파머스 글램핑" 표현 사용 금지. 허용 표기는 "스마트팜 공간대여", "스마트팜 체험 데크", "스마트팜 공간대여 데크"만.

**Why:** 향재원의 정체성은 일반 숙박/글램핑이 아니라 "농업 체험 공간대여"임. CLAUDE.md에 절대 변경 금지로 명시. 단, 내부 스펙 문서(`memory/specs/farm-package-v1.md`)는 비교 설명용으로 "파머스 글램핑" 단어가 일부 등장 — 이는 외부 카피로 전파되면 안 됨.

**How to apply:** 코드/카피/문서 작성·리뷰 시 "글램핑" grep 결과는 모두 버그로 간주. 내부 스펙·비교 분석에서 경쟁군 설명 외 사용자 노출 텍스트에는 절대 사용 금지.
