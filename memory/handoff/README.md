# 📬 Handoff — 파트별 작업 지시서

독스(기획자/QA) 창이 나머지 3개 파트에 전달하는 작업 지시서 보관 폴더.

## 워크플로

```
[독스/QA] — 이슈 발견 → 파트별 지시서 작성 → memory/handoff/*.md 커밋
    ↓
사용자가 각 창에 "해당 지시서 읽고 작업해" 전달
    ↓
[부킹] / [스마트팜] / [디자인] — 본인 브랜치에서 작업 → commit → push
    ↓
[독스/QA] — git fetch → 검증 → 통과 시 main 병합 검토
```

## 각 파트가 작업 완료 후

각 파트 창은 자기 브랜치에 푸시하고, 사용자에게 "○○ 파트 완료"를 보고.
사용자가 독스 창에 전달하면 독스가 `git fetch` 후 검증.
