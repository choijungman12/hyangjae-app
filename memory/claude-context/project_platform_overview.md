# 🌿 향재원(HYANGJAEWON) AI 스마트팜 통합 운영 플랫폼

> **Why this exists**: 신규 Claude 세션이 향재원 플랫폼의 가치 제안·시장 포지션·핵심 기능을 한 번에 이해하도록. CLAUDE.md가 "무엇을 만들었는가"를 다룬다면, 이 문서는 "왜 만들었고 누구에게 어떤 가치를 주는가"를 다룬다.
>
> **How to apply**: 사용자가 사업 비전·마케팅·투자 자료·앱스토어 문구·언론 인터뷰·정부 지원 신청서 작성을 요청할 때 이 문서를 우선 참조. 코드 작업에는 CLAUDE.md를 우선.

## 1. 개발 계기 — 왜 만들어졌는가

### 배경 문제 의식

**① 농가의 디지털 격차**
국내 스마트팜 보급률은 2026년 기준 약 14%에 불과. 전국 100만 농가 중 90% 이상이 여전히 엑셀·종이·카카오톡으로 시세·재배·자재비용을 따로 관리. **"통합 관리 플랫폼"이 사실상 부재**.

**② 시설 투자 의사결정의 맹점**
스마트팜 신규 진입자가 가장 먼저 부딪히는 벽은 **정확한 비용 산출**.
- 평당 단가가 시설 유형(비닐·트러스·유리·패시브)에 따라 25만~200만 원으로 8배 편차
- 정부 보조금(ICT 60%, 청년농 80%) 적용 후 자부담 계산 복잡
- 수경/수직농장 추가 비용, IoT 레벨별 옵션을 종합 계산하는 도구가 없음

**③ 시세·재배·IoT가 분절된 시장**
- KAMIS 시세는 KAMIS 앱
- 재배 가이드는 농사로
- IoT 센서는 제조사별 폐쇄 앱
- 로봇 제어는 기업용 SCADA
- → 농민이 **앱 5개를 번갈아 봐야 의사결정 1건** 가능

**④ AI 진단의 비현실성**
PlantNet·Plant.id 등 글로벌 AI는 영문 학명만 반환 → 한국 농민이 결과를 해석할 수 없음.

### 설립 의도

> **"수지분석 → 시설설계 → 재배관리 → IoT/로봇 자동화까지, 스마트팜 의사결정 전 과정을 단일 앱으로 통합한다."**

운영사 팜스마트(대표 최정만)는 자체 시범 단지(서울 양재동 178-4, 536평)를 **리빙랩**으로 활용하면서 그 안에서 굴러가는 KAMIS·IoT·로봇·AI 기능을 그대로 앱으로 노출 — **농민이 무료로 쓰는 SaaS**로 개방하는 것이 핵심 철학.

## 2. 전체 시장 분석

### 글로벌 스마트팜 시장
| 지표 | 값 |
|---|---|
| 2025 세계 시장 | 약 290억 달러 |
| 2030 예상 | 510억 달러 (CAGR 12%) |
| AI 농업 분야 | 2030년까지 100억 달러 돌파 |
| 농업 로봇 시장 | 2030년 350억 달러 (CAGR 18%) |

### 국내 스마트팜·농업 IT 시장
| 지표 | 값 |
|---|---|
| 2026 국내 스마트팜 시장 | 약 2조 1천억 원 |
| 정부 목표 | 2030년까지 보급률 30% |
| ICT 융복합 보조사업 예산 | 연 약 6,000억 원 |
| 스마트팜 청년농 지원 | 자부담 20% (보조 80%) |

### 작물 데이터·시세 공공 인프라
- **KAMIS** (농수산물 유통정보): 14종 이상 작물 도매·소매·경매가 일별 제공
- **농사로(RDA)**: 표준영농교본·재배 매뉴얼
- **OpenAPI 가락시장**: 실시간 경매가
- → 모두 무료 개방되어 있으나 **API 연결·해석 능력 부족**으로 활용도가 낮음

### 농업 자동화 로봇 시장
- 협동 로봇(Cobot) 도입 농가 연 30% 증가
- Dobot CR10·UR5e·OMRON TM 등 6축 로봇 가격 2,000만~5,000만 원대로 진입 장벽 하락
- 수확·세척·포장 자동화 수요 급증 (인건비 상승, 농촌 고령화)

## 3. 한국 시장에서의 농업 활용성

### ① 스마트팜 신규 진입자의 의사결정 도구
시설 평수·작물·자동화 수준만 입력하면 **총 투자비용·연 매출·BEP·자부담**까지 자동 산출. 청년농 보조금 80%까지 반영해 **"내 돈 얼마 들어가는지"가 한 화면에서 확인** 됨.

### ② 기존 농가의 운영 효율 향상
- 14종 작물 KAMIS 실시간 시세 자동 갱신 (24시간 캐시)
- 작물별 5개년 ROI·BEP 시뮬레이터
- 영농 일지·작업 캘린더·푸시 알림으로 **"잊어버려서 놓치는 작업" 제로화**

### ③ IoT 장비 호환성 문제 해결
**QR 코드 한 번 스캔으로 IoT 장비 연결** — 제조사별 폐쇄 앱 거치지 않고 표준화된 화면에서 통합 모니터링/제어. HTTP REST·MQTT over WebSocket·Mock 3가지 프로토콜 지원.

### ④ AI 식별의 한국화
PlantNet API(46만종) 결과를 **자체 한글 매핑 테이블 7단계** 로 변환:
- 학명 75종, 속(Genus) 55개, 영문 common name 69개 매핑
- 예: *Anthurium* → **안스리움 (Flamingo Flower)**
- 비전문 농민도 결과 즉시 해석 가능

### ⑤ 협동 로봇 농업 도입의 실증 모델
Dobot CR10 6축 로봇을 앱에서 직접 제어 — 작물 스캔→AI 판정→수확→세척→포장 자동 시퀀스(약 15초). **로봇 도입 망설이는 농가가 시뮬레이션·테스트 가능**.

### ⓵ 💰 수지 분석 (Profitability)

#### 화면 구성
| 페이지 | 용도 |
|---|---|
| `/profit-analysis` | 작물 1종 단위 ROI·BEP 계산기 |

#### 핵심 기능
- **실시간 시세 연동 매출 계산**: KAMIS API + 가락시장 OpenAPI 직결 → 작물별 도매·소매·경매가 자동 반영
- **세분화 비용 입력**: 전기·인건비·배양액·묘목·기타 5개 항목 연 단위 분리
- **3대 시나리오 동시 계산**: 낙관·현실·비관 — 수확률·예약률 가변
- **고추냉이 5개년 모델**: 잎·근경 판매가 차등(잎 도매/B2B/직판, 근경 도매/가공/고급소매), 보조금 9천만 + 융자 2억 투자구조 반영, 누적 손익 손익분기 연도 자동 산출
- **통합 시뮬레이터 KPI 4종**: 총 시설투자비·자부담·연간 순이익·ROI·회수기간 → 슬라이더 조작 즉시 반영
- **매출 분해 바차트**: 작물 매출·제품(가공) 매출·기타 매출 시각화

#### 사용 데이터
- `src/lib/kamisApi.ts` — KAMIS 14종 (24시간 캐시 + Mock fallback)
- `src/lib/garakApi.ts` — 가락시장 OpenAPI
- `src/data/crops.ts`·`cropMarketInfo.ts` — 14종 기본 시세·시장 정보

#### 차별점
- **공공데이터 + 사용자 입력 하이브리드** — 시장 변동을 따라가면서도 농가별 실제 비용 반영
- 정부 보조금·융자 자동 차감 → **"실제 내 자부담"** 표시
- 5개년 누적 차트로 BEP·투자회수 시점 시각화

- **건강도 점수**: 색상 분석(녹색/갈변/황화/적색 비율) 기반 0~100 점수
- **잎 크기 자동 측정**: 카메라 25cm 시야각 기반 cm 단위 추정
- **AI 챗봇 영농 상담** — 작물·날씨·시장 데이터 종합

#### 차별점
- 시설재배(스마트팜) **+** 노지재배 동시 제공
- 고추냉이 18~24개월 장기 재배 단독 가이드 (국내 최초 수준)

- 초보자 안내문 + 전문가용 상세 데이터 **2층 레이어**

### ⓷ 🤖 IoT 및 로봇 연결 (IoT & Robot)

#### 화면 구성
| 페이지 | 용도 |
|---|---|
| `/smart-farm-data` | 실시간 센서 통합 대시보드 |
| `/device-control` | QR 스캔 IoT 연결 + **Dobot CR10 6축 로봇 제어** |

#### 지원 센서·장비
- **센서 7종**: 온도 · 습도 · 토양 수분 · CO₂ · 광량 · pH · EC
- **제어기(액추에이터) 6종**: 환기팬 · LED 조명 · 관수 펌프 · 난방기 · 냉방기 · CO₂ 공급기
- **협동 로봇**: Dobot CR10 (페이로드 10kg, TCP/IP 9090)

#### 핵심 기능
- **QR 스캔 즉시 연결** — JSON·커스텀 스키마·HTTP·IP 모두 파싱 (`src/lib/iotClient.ts`·`qrScanner.ts`)
- **3종 프로토콜 지원**: HTTP REST · MQTT over WebSocket(Mosquitto) · Mock(데모용)
- **실시간 5초 폴링** + 수동 새로고침
- **Dobot CR10 6축 관절 제어** — J1~J6 ±5° 단위, 속도 1~100% 조절
- **자동 수확 시퀀스** — 스캔→측정→AI 판정→수확→픽업→세척→복귀 (약 15초)
- **8가지 사전 정의 명령** — 홈 복귀·작물 스캔·수확·픽업·세척·포장·일시정지·재개
- **명령 로그 50건 보존** with 타임스탬프
- **로컬 네트워크 CORS 안내** — 장비 펌웨어 설정 가이드 동봉

#### 사용 데이터
- `src/lib/iotClient.ts` — 연결 관리 + 명령 전송
- `src/lib/qrScanner.ts` — 카메라 API 기반 QR 스캔
- localStorage — 연결 정보 영구 저장

#### 차별점
- **제조사 종속 없는 표준화 인터페이스** — 어떤 IoT 장비든 QR/수동 입력으로 연결
- **로봇 제어 UI 통합** — SCADA·티칭펜던트 없이 모바일에서 직접
- **자동화 파이프라인** — 센서 → AI 판정 → 로봇 동작이 한 흐름으로 묶여 무인 운영 가능
- **장비 보호 경고 배너** 상시 노출 — 안전성 우선

- **시설+IoT+재배방식+보조금 일괄 산출**
- **항목별 비용 분해**: 구조비·설비비·전기공사·인허가·예비비
- **평당/㎡당 단가 자동 재계산** — 규모에 따른 단가 변동 반영
- **데이터 출처 투명 공시** — 농림부 표준 단가, H빔 시공업체 평균, 정부 지원사업 기준 명시
- **시세 트렌드 14종 그래프** — KAMIS 데이터 시각화로 작물 선택 의사결정 지원

#### 사용 데이터
- `src/lib/kamisApi.ts` — 14종 시세
- `src/data/cropMarketInfo.ts` — 작물별 시장 정보(산지·시즌·변동성)
- 농림부 표준 기준단가, H빔 시공업체 평균 견적, 정부 지원사업 기준

#### 차별점
- **"건축비 + IoT + 보조금"** 종합 견적 도구가 시중에 사실상 없음
- 청년농 우대까지 자동 반영 → 청년 진입 장벽 낮춤
- 출처 투명 공시로 **법적 근거 확보**

## 5. 사회적 역할과 비전

### 🇰🇷 사회적 역할

**① 스마트팜 디지털 격차 해소**
시설 견적·시세·재배·IoT·로봇 제어를 **앱 하나로 무료 제공**. 영세·고령 농가도 대기업 수준 데이터에 접근.

**② 농업 청년 진입 장벽 완화**
보조금 80% 자동 반영, 5개년 ROI 시뮬레이션, 단계별 가이드 → **"막연한 두려움"을 정량적 의사결정으로 전환**.

**③ 공공데이터 활용 모델 제시**
KAMIS·농사로·가락시장 OpenAPI를 한 앱에 통합한 **레퍼런스 케이스**. 다른 농업 스타트업이 그대로 차용 가능 (오픈 GitHub).

**④ 농업 로봇·AI 보급의 실증 거점**
Dobot CR10 협동 로봇과 PlantNet AI의 실제 농업 적용 사례를 공개 — 도입 망설이는 농가가 동작 영상·시퀀스 확인 후 도입 가능.

**⑤ 식량 주권 기여**
국내 고추냉이 수입 의존도(약 90%)를 낮추는 **국산화 모델**. 18~24개월 장기 재배 데이터를 표준화해 후속 농가에 공유.

**⑥ 농촌 인구 고령화·인력난 대응**
6축 로봇 자동 수확·세척·포장 시퀀스로 **인건비 절감 + 노동 강도 경감**. 1인 농가 운영 가능성 제시.

### 🚀 비전

> **"농업의 모든 의사결정을 데이터로, 모든 작업을 자동화로."**

| 단계 | 시기 | 목표 |
|---|---|---|
| **Phase 1** | 2026~2027 | 양재 시범 단지 정상 운영, 14종 작물 + Dobot CR10 실증 완료, GitHub 오픈 |
| **Phase 2** | 2028~2029 | Supabase 백엔드 정식 연동, MQTT 실 환경 적용, 100개 농가 파일럿, KAMIS 프록시 SaaS화 |
| **Phase 3** | 2030~ | 전국 1,000개 농가 사용, AI 컨설팅 SaaS 분리 출시, 농업 로봇 표준 인터페이스 제안, 동남아·일본 수출 |
